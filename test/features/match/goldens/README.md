# Match goldens

These three PNG files are generated by `match_golden_test.dart` using the same
deterministic harness as the Discovery goldens. The viewport and device-pixel
ratio are fixed, reveal animations are disabled, Material Icons are loaded,
and remote profile media is replaced with in-memory images. No network request
is made.

Regenerate intentionally from the repository root:

```powershell
flutter test --no-pub --update-goldens test/features/match/match_golden_test.dart
```

Review every changed PNG before committing. Ordinary test runs compare against
the checked-in files and must not pass `--update-goldens`.
