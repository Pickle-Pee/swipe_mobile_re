# Chat goldens

The ten PNG files in this directory are generated by
`chat_golden_test.dart`. The harness fixes the logical viewport to 390 x 844
at device-pixel ratio 1, disables animations, uses the Midnight Aura theme,
and replaces remote avatars and message media with deterministic in-memory
images. It never performs network requests.

Regenerate intentionally from the repository root:

```powershell
flutter test --update-goldens test/features/chat/chat_golden_test.dart
```

Review every changed PNG before committing. Ordinary test runs compare against
the checked-in files and must not pass `--update-goldens`.
