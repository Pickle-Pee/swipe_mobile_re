# Discovery goldens

The six PNG files in this directory are generated by
`discovery_golden_test.dart`. The harness fixes the logical viewport to
390 x 844 at device-pixel ratio 1, disables animations, uses the app's
Midnight Aura theme, and replaces remote media with an in-memory 2 x 2 RGBA
image.
It never performs network requests.

Regenerate intentionally from the repository root:

```powershell
flutter test --update-goldens test/features/discovery/discovery_golden_test.dart
```

Review every changed PNG before committing. Ordinary test runs compare against
the checked-in files and must not pass `--update-goldens`.
