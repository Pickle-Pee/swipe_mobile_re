# Public profile goldens

These five PNG files are generated by `public_profile_golden_test.dart` using
the existing Discovery golden approach. The harness fixes the viewport and
device-pixel ratio, disables animations, loads Material Icons, and substitutes
remote media with deterministic in-memory images. It performs no network
requests.

Regenerate intentionally from the repository root:

```powershell
flutter test --no-pub --update-goldens test/features/profile/public_profile_golden_test.dart
```

Review every changed PNG before committing. Ordinary test runs compare against
the checked-in files and must not pass `--update-goldens`.
